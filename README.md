## Getting Started

### Option 1: Use the Deployed Version
Just visit the live site at `tafara-ai.vercel.app` (once deployed)

### Option 2: Run Locally

1. Clone this repository:
```bash
git clone https://github.com/YOUR-USERNAME/tafara-ai.git
cd tafara-ai
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## How It Works

1. **Get an API Key**: Use the shared key (with login) or get your own free OpenRouter API key
2. **Configure Your AI**: Set personality, behavior, and instructions in plain English
3. **Start Chatting**: Your custom AI is ready to use!

## Getting Your Own API Key

1. Go to [OpenRouter.ai](https://openrouter.ai)
2. Sign up for a free account
3. Navigate to the "Keys" section
4. Create a new API key
5. Copy and paste it into Tafara.ai
